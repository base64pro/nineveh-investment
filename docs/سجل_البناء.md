- **الواجهة:** Next.js 15 (App Router) · TypeScript **strict** (`noUncheckedIndexedAccess`) · Tailwind v3.4 · shadcn/ui · Framer Motion v11 · RTL · خط **Readex Pro**.

- **العرض (deck.gl `MapboxOverlay`، interleaved:false، فوق م1):**

  - `IconLayer` (`parcel-markers.ts`، canvas، حجم ثابت) — **إشارة دبوس لكل قطعة:** ساق أبيض + حلقة بيضاء تحاوط القرص + لبّ بلون الحالة. **كل الإشارات ظاهرة دائماً** (أُزيل `CollisionFilterExtension` بقرار المستخدم — لا إخفاء/تذبذب).
- **الرسم والربط:** `terra-draw` (+ محوّل maplibre) — زرّ عائم، مضلّع بنفسجي؛ عند الإنهاء:
  - استنتاج مكاني `src/features/map/lib/spatial-inference.ts` (`inferName`: turf centroid ∩ booleanPointInPolygon → `name_ar`).
  - فرع: قطعة جديدة → RPC `create_assumed_parcel(p_geom,p_district,p_subdistrict)` + فتح نموذج المفترضة؛ أو ربط بسجل قائم → RPC `create_parcel_geometry(p_geom,p_parcel_no,p_muqataa_no)`. كلاهما `ST_GeomFromGeoJSON→ST_Multi`، security invoker.

- **في البطاقات:** زرّ **«الموقع المباشر»** (`src/components/ui/live-location-button.tsx`، أبيض دائري، `stopPropagation`) بجانب الاسم (المفترضة) أو القطاع (الفرص/الرخص) → `requestFlyTo`؛ وزرّ **«ارسم»** (عند وجود رقم قطعة) → `requestStartDraw`.
- **لوحة تبديل الطبقات** (الحدود/القطع) أزرار عائمة يمين الخريطة + زرّ «كامل نينوى».
