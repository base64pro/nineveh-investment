## الحزمة التقنية (ثابتة)
- **الواجهة:** Next.js 15 (App Router) · TypeScript **strict** (`noUncheckedIndexedAccess`) · Tailwind v3.4 · shadcn/ui · Framer Motion v11 · RTL · خط **Readex Pro**.
- **الخريطة:** MapLibre GL JS + MapTiler (متّجه عبر وسيط خادمي) + **deck.gl** (`@deck.gl/core`،`/layers`،`/mapbox`،`/extensions`) + **terra-draw** (+ محوّل maplibre) + **@turf** (centroid · booleanPointInPolygon · bbox).
- **القاعدة:** Supabase (PostgreSQL + PostGIS + RLS «المصادَقون فقط») عبر `@supabase/ssr`؛ الهجرات بـ`scripts/migrate.ts` (تتبّع `schema_migrations`، `npm run migrate`)؛ تحقّق بـ`pg`.
- **الحالة/المزامنة:** TanStack Query (قراءات) + Server Actions (كتابات) + Supabase Realtime (إبطال).
- **الذكاء (م4):** Anthropic عبر الخادم فقط · RAG على pgvector.

### م2.4 — تلوين/رسم/استنتاج/تنقّل القطع
بنية الخريطة المتقدّمة (المكوّن المركزي `src/features/map/components/investment-map.tsx` ~600 سطر):
- **المصدر:** view `map_parcels` (هجرة `..120003` ثم `..120007`) — يوحّد المفترضة (من `assumed_parcels.geom`) والفرص/الرخص (من `parcel_geometry` بالمطابقة على `parcel_no`+`muqataa_no`)، ويُخرِج: `kind · ref_id · parcel_no · state · geometry (GeoJSON عبر st_asgeojson) · label`. **تنبيه:** أعمدة PostGIS تُقرأ EWKB عبر REST ⇒ الـview ضروري لإخراج GeoJSON.
- **القراءة:** `src/features/map/lib/use-map-parcels.ts` (مفتاح `["map_parcels"]`) → `FeatureCollection`. الألوان: `parcel-colors.ts`.
- **العرض (deck.gl `MapboxOverlay`، interleaved:false، فوق م1):**
  - `GeoJsonLayer` — ملء شفّاف + حدّ + **هالة هولوكرامية**؛ المحدّدة تمتلئ بلون/شدّة الحدّ (ألفا≈232) وتخفت ما حولها.
  - `IconLayer` (`parcel-markers.ts`، canvas، حجم ثابت) — **إشارة دبوس لكل قطعة:** ساق أبيض + حلقة بيضاء تحاوط القرص + لبّ بلون الحالة. **كل الإشارات ظاهرة دائماً** (أُزيل `CollisionFilterExtension` بقرار المستخدم — لا إخفاء/تذبذب).
- **الرسم والربط:** `terra-draw` (+ محوّل maplibre) — زرّ عائم، مضلّع بنفسجي؛ عند الإنهاء:
  - استنتاج مكاني `src/features/map/lib/spatial-inference.ts` (`inferName`: turf centroid ∩ booleanPointInPolygon → `name_ar`).
  - فرع: قطعة جديدة → RPC `create_assumed_parcel(p_geom,p_district,p_subdistrict)` + فتح نموذج المفترضة؛ أو ربط بسجل قائم → RPC `create_parcel_geometry(p_geom,p_parcel_no,p_muqataa_no)`. كلاهما `ST_GeomFromGeoJSON→ST_Multi`، security invoker.
- **التنقّل (pub/sub):** `src/features/map/lib/map-nav-store.ts` — `requestFlyTo/onFlyTo` · `requestOpenParcelForm/onOpenParcelForm` · `requestOpenParcelDetail/onOpenParcelDetail` · `requestStartDraw/onStartDraw`. النوافذ العامّة `assumed-modals.tsx` في القشرة تستمع للأحداث.
- **نقر القطعة:** deck `pickObject` → تحديد/تظليل (`autoHighlight`)؛ نقر الفراغ يلغي. **نقر الإشارة:** نافذة عمودية أنيقة بجانب القرص — **عنوان القطعة** (`label`) ثم زرّ «عرض» (→`requestOpenParcelDetail`) ثم زرّ «الموقع» (→`requestFlyTo`).
- **في البطاقات:** زرّ **«الموقع المباشر»** (`src/components/ui/live-location-button.tsx`، أبيض دائري، `stopPropagation`) بجانب الاسم (المفترضة) أو القطاع (الفرص/الرخص) → `requestFlyTo`؛ وزرّ **«ارسم»** (عند وجود رقم قطعة) → `requestStartDraw`.
- **لوحة تبديل الطبقات** (الحدود/القطع) أزرار عائمة يمين الخريطة + زرّ «كامل نينوى».
