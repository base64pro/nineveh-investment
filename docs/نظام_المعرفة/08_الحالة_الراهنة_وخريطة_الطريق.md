## ج. البنية وتدفّق البيانات

- التموضع مرسىً عند مركز القطعة (turf centroid). **مصدر التحويل مختلف بين المسارين** (تمييز دقيق): النماذج **المرفوعة** تُموضَع عبر عمود `transform` (jsonb: scale/rotationDeg/elevationM/anchorOverride) على جدول `parcel_models`؛ أمّا النماذج **البارامتريّة** فتُموضَع عبر أعمدة منفصلة على جدول `parcel_parametric_models`: `rotation_deg` (**int NOT NULL default 0**) و`width_m`/`depth_m`/`height_m` (numeric **nullable**، فارغ = اشتقاق تلقائيّ من البصمة) — أُضيفت في الهجرة `20260624140000`. الأبعاد/التوجيه يدخلان مفتاح التخبئة.
- التنقّل عبر `map-nav-store.ts` (pub/sub)؛ الجولة م9.5 ستضيف `requestAutoTour/stop`.

## هـ. الثوابت غير القابلة للتفاوض (ذات الصلة بالمجال)

2. **لا `next build` وdev يعمل**؛ الخادم المحلي = Start-Process مفصول على :3000، سجلّه `.next/start.log`.

4. **§ج.9 حتمي** — بطاقة الجاهزية (م9.4) دالّة نقية + اختبارات، لا ذكاء.
5. **الأدوار بـRLS/is_admin()** على القاعدة؛ `useRole` إخفاء مكمّل.
6. تعديلات الجوال بصفر مساس بـmd+/المنطق/الأدوار/RLS.
