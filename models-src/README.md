# models-src — مصادر نماذج 3D الخام (عالية الدقّة)

ضع هنا ملفّات المجسّمات **الخام** (glb/gltf) كمصدر تحريريّ — قبل التحسين للويب.

- **لا تُلتزَم ولا تُدفَع:** ملفّات `*.glb`/`*.gltf` هنا متجاهَلة في `.gitignore` (قد تتجاوز حدّ GitHub 100MB، ولا داعي لرفعها).
- **المُخرَج المُحسَّن للويب** يُولَّد آلياً ويُكتَب إلى `public/models/` (هو الذي يُخدَم ويُلتزَم).

## التحسين (بلا بلندر — خطّ معالجة آليّ)
1. ضع الملفّ الخام هنا، مثلاً: `models-src/nineveh-authority.glb`.
2. شغّل: `npm run optimize:model`
   - يُجري: لحام رؤوس + ضغط هندسة Draco + تبسيط دقيق بحدّ خطأ منخفض (يحفظ الشكل) + قوام 2K مضغوط.
   - يكتب: `public/models/nineveh-authority.glb` (~٢٠–٣٠ ميجا، مطابق بصرياً من مسافة الجولة).

> يُضبَط حدّ التبسيط/القوام حسب الحجم المستهدَف مع الحفاظ على أمانة الشكل.
